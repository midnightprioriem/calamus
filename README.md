### :running: Run Locally

Clone the project

```bash
  git clone https://github.com/midnightprioriem/calamus.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  yarn install
```

Start the app

```bash
  yarn start
```
